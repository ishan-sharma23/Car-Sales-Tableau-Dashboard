# Data Dictionary — Car Sales Dataset

| Field Name | Data Type | Description |
|---|---|---|
| Car_id | String | Unique identifier for each car sale |
| Date | Date | Sale transaction date |
| Customer Name | String | Name of the buyer |
| Gender | String | Buyer gender (Male / Female) |
| Annual Income | Number | Annual income of the buyer |
| Dealer_Name | String | Name of the dealership |
| Company | String | Car manufacturer / brand |
| Model | String | Car model name |
| Engine | String | Engine type |
| Transmission | String | Manual / Auto |
| Color | String | Car colour |
| Price ($) | Number | Sale price in USD |
| Dealer_No | String | Dealer reference number |
| Body Style | String | SUV, Sedan, Hatchback, Passenger, Hardtop |
| Phone | String | Buyer contact number |
| Dealer_Region | String | Geographic region of the dealer |
