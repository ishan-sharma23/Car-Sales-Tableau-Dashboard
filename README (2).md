# Dashboard Files

Place your Tableau packaged workbook here:

- `Car_Sales_Dashboard.twbx` — Main Tableau packaged workbook (includes data source)

## How to open
1. Install Tableau Desktop or Tableau Public (free)
2. Double-click the `.twbx` file, or go to File → Open

## Requirements
- Tableau Desktop 2021.x or later (recommended)
- Tableau Public (free) also works for viewing
